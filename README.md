# Better Decompiler

Better Decompiler is a script designed to enhance the default decompiler by improving output readability, structure, and overall code clarity. It delivers cleaner, more organized results compared to standard decompilation tools while maintaining performance and usability.

## Status

Better Decompiler is currently in **beta**.  
Some features may be unstable, incomplete, or subject to change as development continues.

## Features

- Improved decompiled output formatting  
- Cleaner structure and enhanced readability  
- Custom built-in user interface  
- Designed for scalability and future expansion  
- Performance-focused processing  
- Organized output restructuring  

## Current Limitations

- No native function support yet  
- No Dex integration  
- Advanced optimization passes are still in development  

## Project Goal

The goal of Better Decompiler is to significantly improve decompiled script quality while keeping the system efficient and extensible. The project aims to evolve into a professional-grade decompilation enhancement tool.

## Notes

This project is actively being developed. Feedback, testing, and suggestions are welcome to help refine and expand its capabilities.
